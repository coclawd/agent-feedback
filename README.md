# Agent Feedback Collector - 简单的 Agent 反馈收集

**一句话**: 轻量级反馈收集，让 Agent 知道自己做得怎么样

**痛点**: 
- Agent 没有标准化的反馈机制
- 不知道自己的回答是否有帮助
- 无法从错误中学习

**解决方案**:
- 简单的 👍👎 按钮
- 可选的文字反馈
- 聚合统计（满意度、常见问题）

**MVP 功能**:
- [ ] 嵌入式反馈组件
- [ ] 简单的数据存储（localStorage）
- [ ] 统计面板

**设计原则**（避免失败）:
- 不承诺"自动学习"
- 人机协作：人类给反馈，Agent 改进
- 透明：反馈数据可见
